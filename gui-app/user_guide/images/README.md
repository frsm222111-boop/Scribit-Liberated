## Image Guidelines

- **Format:** PNG with transparent background (where appropriate)
- **Resolution:** At least 1280px wide for main screenshots
- **Annotations:** Use red arrows or boxes to highlight important UI elements
- **Consistency:** Use the same app theme/appearance across all screenshots
- **Privacy:** Don't include personal information (IP addresses can be 192.168.1.100 as example)
- **File size:** Optimize images to keep file sizes reasonable (< 500KB each)

## Taking Screenshots

### macOS
- **Full window:** Cmd + Shift + 4, then Space, click window
- **Selection:** Cmd + Shift + 4, drag selection
- **Edit:** Open in Preview, use Markup tools to add arrows/boxes

### Windows
- **Full window:** Alt + Print Screen
- **Selection:** Win + Shift + S
- **Edit:** Use Paint or Snipping Tool to add annotations
